# Segurança

O FieldServiceSuite prioriza diagnóstico e relatório. Alterações de configuração devem exigir confirmação explícita do técnico.

Boas práticas:

- Não armazenar credenciais.
- Não coletar dados pessoais desnecessários.
- Não executar comandos destrutivos automaticamente.
- Registrar ações relevantes em log.
- Informar limitações quando a execução não possuir privilégios administrativos.
